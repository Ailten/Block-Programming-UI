Explication

Elements

Le retangle gris est la zone de travaille (canvas). Le rectangle gris foncé (a gauche) est le menu. Le menu contient les type de block que tu peu utiliser. Le rectangle "poubelle" en haut du menu est la corbeille pour détruire les block don tu n'a plus besoin.

Block

Les block du menu peuvent etre placé dans la zone de travaille (clicker-glisser), pour réer une copie "utilisable". Les block peuvent etre connecté les un en dessous des autres (sauf les block evenement en orange, qui doivent etre en premier). Pour connecter un block A en dessou d'un block B, clicker-glicer A sur B. Certain block peuvent contenire des parametres modifiable, ou encore d'autre block a l'intérieur. L'ordre des block sont important, il executerons leurs instruction du haut vers le bas. Pour suprimer un block dont tu n'a plus besoin, clicker-glicer dans la poubelle (du menu).
